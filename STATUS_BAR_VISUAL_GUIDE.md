# Визуальное Руководство по Статус Бару

## Внешний Вид Статус Бара

### 1. Статус: Ожидание (Waiting)
```
┌──────────────────────────────┐
│   Статус: Ожидание...        │  ← Текст белый
└──────────────────────────────┘
         Фон: #FFA500 (оранжевый)
```
**Когда отображается**: В момент инициализации подключения к серверу

### 2. Статус: Подключено (Connected)
```
┌──────────────────────────────┐
│   Статус: Подключено         │  ← Текст белый
└──────────────────────────────┘
         Фон: #4CAF50 (зелёный)
```
**Когда отображается**: После успешного подключения к серверу

### 3. Статус: Ошибка (Error)
```
┌──────────────────────────────┐
│   Статус: Ошибка             │  ← Текст белый
└──────────────────────────────┘
         Фон: #F44336 (красный)
```
**Когда отображается**: При возникновении ошибки подключения

### 4. Статус: Таймаут (Timeout)
```
┌──────────────────────────────┐
│   Статус: Таймаут            │  ← Текст белый
└──────────────────────────────┘
         Фон: #FF9800 (оранжевый тёмнее)
```
**Когда отображается**: При превышении времени ожидания ответа от сервера

### 5. Статус: Разрыв связи (Disconnected)
```
┌──────────────────────────────┐
│   Статус: Разрыв связи       │  ← Текст белый
└──────────────────────────────┘
         Фон: #9E9E9E (серый)
```
**Когда отображается**: При потере соединения или намеренном отключении

## Размеры и Стиль

### Статус Бар
- **Ширина**: минимум 250px, автоматическое растяжение
- **Высота**: автоматическая (зависит от padding)
- **Padding**: 10px со всех сторон
- **Font-size**: 14px
- **Font-weight**: normal (не жирный)
- **Font-color**: white (белый)
- **Border-radius**: 5px (скруглённые углы)
- **Text-align**: center (по центру)

### Кнопка Выхода
- **Padding**: 10px
- **Font-size**: 14px

## Поведение

### При запуске игры
1. Статус бар **скрыт** на экране приветствия
2. При нажатии "Начать игру":
   - Появляется статус "Ожидание..." (оранжевый)
   - Через 1 секунду меняется на "Подключено" (зелёный)

### Во время игры
- Статус бар **видим** всё время
- Отображает текущее состояние подключения
- Автоматически обновляется при изменении статуса

### При выходе из игры
- Статус меняется на "Разрыв связи" (серый)
- После возврата на экран приветствия статус бар **скрывается**

## Адаптивность

Статус бар использует `addStretch()` между собой и кнопкой выхода:
- На узких экранах: минимальная ширина 250px
- На широких экранах: растягивается для заполнения пространства
- Кнопка выхода всегда справа

## Код Цветов

### Цветовая палитра

| Hex код | RGB             | Название           | Использование |
|---------|-----------------|-------------------|---------------|
| #FFA500 | rgb(255,165,0)  | Оранжевый         | Waiting       |
| #4CAF50 | rgb(76,175,80)  | Зелёный (Material)| Connected     |
| #F44336 | rgb(244,67,54)  | Красный (Material)| Error         |
| #FF9800 | rgb(255,152,0)  | Оранжевый тёмнее  | Timeout       |
| #9E9E9E | rgb(158,158,158)| Серый (Material)  | Disconnected  |

Все цвета взяты из Material Design палитры для единообразия с современными UI.

## Доступность

- **Контрастность**: Белый текст на цветном фоне обеспечивает хорошую читаемость
- **Размер текста**: 14px достаточно большой для комфортного чтения
- **Цветовая кодировка**: Дополняется текстовым описанием для людей с нарушениями цветовосприятия
- **Padding**: Достаточное пространство вокруг текста для удобства чтения

## Интеграция с Темой

Статус бар гармонично вписывается в общий стиль игры:
- Скруглённые углы (border-radius: 5px) согласуются с современным дизайном
- Цвета из Material Design палитры
- Размер шрифта совпадает с размером кнопок (14px)
- Padding идентичен кнопкам (10px)
